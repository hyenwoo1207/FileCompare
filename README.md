# (C# 코딩) 파일 비교 툴(File Compare)
## 개요
- C# 프로그래밍 학습
- 1줄 소개: 파일을 비교하는 프로그램
- 사용한 플랫폼:
- C#, .NET Windows Forms, Visual Studio, GitHub
- 
- 사용한 컨트롤:
- Label, TextBox, ListBox, Button, Listview
- 
- 사용한 기술과 구현한 기능:
- Visual Studio를 이용하여 UI 디자인
- 파일을 비교하는 로직 구현
- ListBox와 ListView를 이용하여 파일 목록과 비교 결과 표시
